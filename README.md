# Projeto-Sistema-de-Restaurante-Universitario
O projeto consiste em um sistema de gerenciamento que automatiza as operações de um restaurante universitário (RU), desde o cadastro de usuários até o controle de acesso e relatórios de consumo. O projeto foi criado como parte da disciplina de POO com objetivo avaliativo, aplicando conceitos como encapsulamento, herança e polimorfismo.
